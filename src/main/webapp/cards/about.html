About

I am just a software developer who likes what he does and wanted to learn AngularJS, TomEE, REST APIs, and 'BigData' technologies.

To do that I am taking a card game I made originally in C# for my networking class at K-State in 2006. While the original goal was to convert it to Java using Struts2, the original code well it kinda sucked. So this represents a complete re-write!

I plan am using AngularJS for the front end, exposing REST APIs in the backend, and currently using MySQL for database(though I want to switch to Cassandra to learn it).

The plan is to implement some card games -

  • Golf - I'll provide instructions to it later
  • Follow that by Pitch, Spades, and/or Hearts
  • Since this is open source, anybody can contribute and implement what they want. Just note your own work please!

Thanks for reading! And if interested you can get the source from GitHub.
